# Imtiaz Mall Customer Analytics - Electronics Section

## Project Overview
This project analyzes customer behavior in Imtiaz Mall's electronics section to combat declining sales and improve customer retention. The analysis covers the complete data science lifecycle from preprocessing to predictive modeling.

## Business Objectives
- Identify factors driving customer churn
- Segment customers for targeted marketing
- Predict future purchase behavior
- Provide actionable business recommendations

## Technologies Used
- Python 3.8+
- Pandas, NumPy (Data processing)
- Matplotlib, Seaborn, Plotly (Visualization)
- Scikit-learn (Machine Learning)
- Jupyter Notebook

## Key Analyses
1. **Data Preprocessing**: Handling missing values, outlier analysis, feature engineering
2. **Exploratory Data Analysis**: Univariate, bivariate, and temporal analysis
3. **Predictive Modeling**: 
   - Regression (predicting spending)
   - Classification (predicting purchase likelihood)
4. **Customer Segmentation**: K-Means clustering
